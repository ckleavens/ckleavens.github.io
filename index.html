Integritetspolicy för Elpriskollen

Elpriskollen värnar om din integritet. Vi samlar inte in någon data från användare, och den enda behörigheten som appen kräver är internetåtkomst. Detta används enbart för att hämta aktuella elpriser från externa källor.

Vilken data samlar vi in?

Appen samlar inte in eller lagrar någon personlig information eller annan användardata.

Behörigheter

Appen kräver följande behörigheter:

  • Internetåtkomst: Används för att hämta aktuella elpriser från www.elprisetjustnu.se. Ingen information om dig eller din enhet delas eller lagras.

Tredje parter

Appen kommunicerar med en extern API-tjänst för att hämta data, men ingen användarinformation delas med eller skickas till denna tjänst.

Ändringar i denna policy

Vi kan uppdatera denna integritetspolicy om appens funktionalitet förändras. Eventuella ändringar publiceras här och meddelas vid behov.
